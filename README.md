# SerpentAI_hornet
# requires 
* serpentai=2018.1.2
* tensorflow>=1.5
* os=windows7/10
* python=3.6

## Installation
* https://github.com/SerpentAI/SerpentAI/wiki
* install steam
* install game Golden Hornet

## Start
* cd SerpentAI_hornet
* serpent launch hornet
* choice game: START->ARCADE->START ARCADE
* serpent play hornet SerpenthornetGameAgent
